💡 What Should You Upload?
The more relevant documents you add, the smarter your bot becomes. Here's what works best:
1

College Handbook / Regulations

The master document with all rules, code of conduct, dress code, attendance policy, disciplinary procedures.

2

Admission & Fee Documents

Fee structure, scholarship details, admission criteria, required documents, important dates and deadlines.

3

Academic Calendar & Exam Schedule

Semester dates, exam timetables, result dates, holiday list, internal assessment schedules.

4

Course & Syllabus Details

Available programs, subjects offered, credit structure, elective options, lab requirements.

5

Hostel & Transport Rules

Accommodation guidelines, bus routes, hostel fees, warden contact, in-time rules.

6

FAQ Document (if you have one)

Any existing FAQ sheet, brochure, or student guide your college already distributes.

Embed on Your Website
Copy the code below and paste it into your college website's HTML — the chat widget will appear automatically.
📋 Step 1 — Download the Widget File
First, save the chatbot widget file (widget.html) to your web server or hosting. Then use the embed code below.
📋 Step 2 — Paste This Into Your Website
Add this snippet just before the </body> tag on every page you want the chat widget to appear.
<!-- College FAQ Bot Widget --> <iframe id="college-bot-frame" src="./widget.html" style="position:fixed;bottom:24px;right:24px; width:380px;height:580px;border:none; z-index:9999;border-radius:20px; box-shadow:0 8px 40px rgba(0,0,0,0.2);" ></iframe> <!-- On mobile, make it full screen --> <style> @media (max-width: 480px) { #college-bot-frame { width: 100vw !important; height: 100vh !important; bottom: 0 !important; right: 0 !important; border-radius: 0 !important; } } </style>
🌐 Where to Host
You need to host the widget.html file somewhere. Here are your options:
A

Same Server as Website

Easiest. Upload widget.html to the same folder as your index.html. No extra setup needed.

B

GitHub Pages (Free)

Put widget.html in a GitHub repo, enable Pages, and use the public URL. 100% free hosting.

C

Netlify / Vercel (Free)

Drag & drop widget.html onto netlify.com. Get a permanent URL in 30 seconds. Free forever.
